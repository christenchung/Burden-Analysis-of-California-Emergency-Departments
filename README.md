# SQL_California-ER-Burden_Data-Analysis
In this SQL project, I am analyzing the burden of Emergency Departments in California from 2021-2023 using SQL queries and Tableau visualizations. 

The data set used: https://data.chhs.ca.gov/dataset/emergency-department-volume-and-capacity/resource/929362c5-513b-4e89-8a9e-b34834a3004d?view_id=26ff11c1-af4d-4c9a-900c-c7ed3f30cfd4
